GitHub Artifact Viewer

Enter a repository and run number to view the plots from the GitHub Actions artifact.

How to use:

  1. Enter the repository in the format "owner/repository-name"
  2. Enter the GitHub Actions run number
  3. Click "View Plots" to download and display the artifacts

Note: This tool works with public repositories and caches downloaded artifacts for faster subsequent access.
